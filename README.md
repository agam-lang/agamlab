# AGAMLAB — MATLAB-like Interactive Scientific Computing Platform

> Part of the [agam-lang](https://github.com/agam-lang) organization.

## Mission

Provide a MATLAB-class interactive scientific computing environment built natively on top of the Agam language and toolchain.

## Key Areas

- Matrix/Linear Algebra
- Signal Processing
- Statistics
- Plotting/Visualization
- Interactive REPL
- Notebook Interface

## Crates

- `agamlab_core/`
- `agamlab_repl/`
- `agamlab_notebook/`
- `agamlab_plot/`
- `agamlab_matrix/`
- `agamlab_signal/`
- `agamlab_stats/`
- `agamlab_io/`

## Status

This repository is under active development as part of the Agam ecosystem.
